Why we're asking you to do this

Facebook doesn't allow software to post to Marketplace directly — every listing has to come from a real Chrome browser logged in to a real Facebook account. This helper sits quietly in your Chrome and fills the form for you when you click the button.

We're publishing it to the Chrome Web Store this week (waiting on Google's approval). Once it's there, install drops to two clicks. Until then, please follow the steps below — same end result, just a few extra clicks once.

Privacy note: The helper only runs on app.firstfinancialcanada.com and facebook.com/marketplace/*. It doesn't read anything else, doesn't track you, and you can remove it from chrome://extensions any time.
Step 1

Download the helper

One small zip file. Save it somewhere you'll remember (Downloads is fine).

⬇ Download First-Fin Helper (.zip)
~80 KB · safe · audited code
Step 2

Unzip it

Right-click the downloaded file → Extract All (Windows) or double-click (Mac). You'll get a folder called firstfin-extension. Don't delete this folder — Chrome reads from it directly.

Tip: drag the unzipped folder somewhere permanent like Documents — if you delete it later, the helper stops working.
Step 3

Open Chrome's extensions page

Copy this address and paste it into your Chrome address bar:

chrome://extensions

(Chrome won't let websites link directly to this page — you have to paste it manually. Click the box above to copy it.)

Step 4

Turn on Developer Mode + load the helper

  1. Top-right of the Extensions page, flip the Developer mode toggle ON.
  2. Three new buttons appear at the top-left: Load unpacked, Pack extension, Update.
  3. Click Load unpacked.
  4. In the file picker, select the firstfin-extension folder you unzipped in Step 2.
  5. You should now see First-Fin Browser Helper in your extensions list with a teal logo.
Done? You can flip Developer Mode back off if you want — the helper stays installed and works fine either way.
