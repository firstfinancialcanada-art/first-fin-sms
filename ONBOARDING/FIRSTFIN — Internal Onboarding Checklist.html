NEW TENANT ONBOARDING
Step-by-step checklist for onboarding a new dealer — follow in order
Before you start: You need admin access to the FIRST-FIN operator panel and the dealer's email address. The whole process takes about 15 minutes. Have the dealer on the phone if possible.
Phase 1 — Create the Tenant
Open app.firstfinancialcanada.com/admin?token=ChangeMe12345
Click + New Tenant and fill in: dealer email, display name, dealership name
Set a temporary password (e.g., 123456) — dealer changes it during wizard
Set subscription status to Active (or Trial if applicable)
Ask the dealer for their inventory page URL (e.g., www.stampedeauto.com/inventory) — you'll need this for domain lock in Phase 3
Text the dealer their login: "Your FIRST-FIN platform is ready. Go to app.firstfinancialcanada.com and log in with [email] / [password]"
Phase 2 — Prepare the Extension Package
Copy the generic-dealer-package folder (from ONBOARDING folder)
Rename the copy to the dealer's name (e.g., Kenneth - Automaxx)
Open README - START HERE.txt — no edits needed, it's generic
Send the folder to the dealer (email zip, USB, Google Drive, etc.)
Phase 3 — Walk the Dealer Through Setup
Login — Dealer logs into app.firstfinancialcanada.com
Wizard — Dealer completes setup wizard (name, phone number, password)
Extension install — Walk through: chrome://extensions > Developer Mode > Load unpacked > select firstfin-extension folder
Extension login — Click FIRST-FIN icon in toolbar > log in with platform credentials
First scan — Go to dealer's website inventory page > click Scan This Page > wait for all vehicles > Sync with Replace
Lock scraper domain — Back in admin panel, click Domain Lock on the tenant and enter the dealer's inventory URL domain (e.g., stampedeauto.com). This restricts the scraper to only work on that dealer's website.
Verify inventory — Check Inventory tab in platform shows all vehicles with correct prices and mileage
Test FB Poster — Click FB Poster tab > pick a vehicle > click Auto-Fill Facebook > verify form fills correctly
Phase 4 — Admin Verification
Back in admin panel — verify tenant shows ACTIVE with correct inventory count
Verify Twilio number is claimed (if applicable)
Send dealer the Dealer Setup Guide PDF for their records
Upsell: DealerTrack Sync
Once the dealer is live, mention DealerTrack Sync as an add-on: "We can connect your DealerTrack account so contacts and deals flow into your CRM automatically. Takes about 15 minutes to set up."

Requires: dealer's DealerTrack credentials, the DT Bridge desktop app installed on their computer, and a brief setup call.
